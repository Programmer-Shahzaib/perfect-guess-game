    🎯 THE PERFECT GUESS (Python Game)
Description:
The Perfect Guess is a simple number guessing game built using Python.
The program randomly selects a number between 1 and 100, and the player has to guess it with hints provided after each attempt.
________________________________________
      Features:
•	Random number generation 🎲
•	User-friendly input prompts
•	Hints:
	"Higher number please"
	"Lower number please"
•	Tracks number of attempts 📊
•	Replay option 🔁
•	Personalized welcome message 👤
________________________________________
      Technologies Used:
•	Python 🐍
•	Built-in random module
________________________________________
      How to Run the Game:
1.	Make sure Python is installed on your system.
2.	Copy the code into a file named:
       main.py
3.	Run the program:
python main.py
________________________________________
      How to Play:
1.	Enter your name.
2.	The system will generate a random number between 1 and 100.
3.	Start guessing:
	If your guess is too high → you'll see "Lower number please"
	If your guess is too low → you'll see "Higher number please"
4.	Keep guessing until you find the correct number.
5.	The game will show:
	Correct number ✅
	Your guess 🎯
	Number of attempts 📈
6.	Choose to play again or exit.
